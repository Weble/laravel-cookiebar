# Laravel cookie consent dialog

## Installation

```
composer require weble/laravel-cookiebar
```

### Publish config

```
php artisan vendor:publish --provider="Weble\Cookiebar\CookiebarServiceProvider" --tag="cookiebar-config"
```

### Publish view

```
php artisan vendor:publish --provider="Weble\Cookiebar\CookiebarServiceProvider" --tag="cookiebar-config"
```
